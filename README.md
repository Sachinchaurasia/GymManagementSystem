# 🏋️ Gym Management System

A complete **Gym Management System** developed using **PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap**. This project helps gym administrators efficiently manage members, staff, and daily gym operations through a user-friendly web interface.

---

## 🚀 Features

- 👨‍💼 Admin Dashboard
- 👥 Member Management
- 🏋️ Staff Management
- 🔐 Secure Login System
- 📋 Member Registration
- 💳 Membership Management
- 🗄️ MySQL Database Integration
- 📱 Responsive User Interface

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Sachinchaurasia/GymManagementSystem.git
```

### 2. Move Project

Copy the project folder into:

```
C:\xampp\htdocs\
```

### 3. Start XAMPP

Start:

- Apache
- MySQL

### 4. Import Database

- Open phpMyAdmin
- Create a new database
- Import the SQL file from the **DATABASE FILE** folder

### 5. Configure Database

Update database credentials inside:

```
dbcon.php
```

Example:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "gym";
```

### 6. Run the Project

Open your browser:

```
http://localhost/Gym-System/
```
